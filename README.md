# FirstChatBot

A simple React chatbot frontend built with Vite.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown by Vite.

## What this includes

- `src/App.jsx`: chat UI with user and bot bubbles
- `src/styles.css`: basic dark theme styling
- `vite.config.js`: React/Vite config

## Next step

Connect `App.jsx` to a backend API or chatbot service by replacing the placeholder bot reply logic in `sendMessage()`.
